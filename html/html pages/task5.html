Task 5

This task involves evaluating the truth value of the propositional expression given the truth values of the individual atoms. First we loop through the string and add the unique propositional atoms to an array that contains the atoms. Next we enter a loop and ask the user for the truth values of the propositional atoms that we discovered in the previous function. Upon providing the truth values we compute the truth value recursively by travelling to the left most propositional atom and start evaluating in pairs of parent,left child and right child or parent and right child depending on whether the parent operator is a negation or not. Here we also use an integer len variable which makes the looping more efficient. Since the size of the array is very large it is redundant to loop through the entire array when there are no valid propositional atoms in those indices. Therefore, the len variables helps in this so we can exit the for loop once we have looped through the number of propositional atoms found.
