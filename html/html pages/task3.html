Task 3

This task involves recursing through the binary tree formed in the previous task and printing the elements using in-order traversal. This means that it will keep travelling to the left until it encounters a propositional atom in which case it will print the propositional atom then go back to the parent and visit the right child and the continue this recursive process until it goes back up to the root of the binary tree where it will finally exit the recursion and stop printing the statement.
