Task 2

This task involves taking the expression in prefix form and forming a binary tree with the given expression. We use a stack that contains Node pointers where Node is a struct that contains pointers to its left and right children as well as some data which is a character. The function loops through the length of the string and checks a few if else statements to add the character as a node of the tree. There is also a function to print the tree which has been commented out as it was just to test if the code worked. We include Task1.h here and in the process inherit all the functions and global variables from it.
