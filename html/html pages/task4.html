Task 4

This task involves computing the height of the binary tree formed in task 2. We use recursion again and keep travelling to the leftmost child until we encounter a propositional atom. We take the height of a child to be 0 and total height to be number of edges + 1. We return 0 on encountering a propositional atom then go back to the previous call and traverse the right side and save the resulting heights of the left and right branches in two variables. We now compare these variables and return the maximum of the two as we want the height of the tree as a whole.
