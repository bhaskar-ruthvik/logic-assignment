Task 1

This task involves converting an infix input to a string in the prefix notation. We are going to provide a well-formed infix formula with the appropriate parenthesis which will first be reversed using a for loop and string operations. Then this reversed string will be passed into a function which uses a stack class to convert the reversed string into its postfix form by comparing the operator precedences. We take this string and reverse it back and we now obtain the final string in prefix form which is the end goal of this task. A template was used to make the stack class in order to accomodate for any other types of stacks that might be needed in the upcoming tasks. The stack was implemented using a linked list with the head of the linked list acting as the top of the stack. The stack methods were implemented abiding by the Time-complexity definitions of the inbuilt style class.
