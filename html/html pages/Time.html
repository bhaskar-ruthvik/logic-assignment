Time Analysis of Code

The following tests were performed on an M1 Apple Macbook Air with 8GB RAM and 256GB Storage. The IDE used for these tests is Visual Studio 2022 for Mac. The outputs produced were recorded and the results attached below are the findings of the analysis. The variables in this were No of unique propositional atoms, The height of the parse tree and the time taken for the execution in millseconds. In order to make this a fairer analysis the truth values were kept constant with all being assigned to 1 as changing the truth values leads to different times as seen in the output testing done previously. This data was recorded in a csv file and was analysed in a jupyter notebook attached in the analysis folder.

Some graphs that depict the relations between the chosen variables:

This is a countplot showing the counts of the time readings we have obtained. We can see that most of the readings obtained are around 0.1ms followed by 0.3ms. The mean time(ms) obtained for all the readings has been calculated as 0.26ms.

This is a scatterplot between the height of the tree and the time taken for execution. We can clearly see that there is a trend of direct proportionality between the two variables. As we can draw a line of bestfit that covers many values at once.

This is a heatmap with the correlation between each of the time variables. As we could observe from the graph te correlation between height of the parse tree and Time in ms is 0.88 which is a very high correlation. Surprisingly, the correlation between Number of unique propositional atoms and height of the tree is not very high. This is because as we have seen in the readings table the number of unique propositional atoms can be few but if the same atoms are repeated again and again the height of the tree increases drastically. Due to the same reason the correlation between time and the number of unique propositional atoms is also not very strong.

If we notice the trend with fixed height and no of unique propositional atom values we can clearly see that the scatterplot has an upward trend showing there is atleast a good amount of correlation between all 3 variables together.
